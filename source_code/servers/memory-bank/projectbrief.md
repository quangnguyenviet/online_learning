# Project Brief: Online Learning Platform (Vitube)

## Project Overview
Vitube is a comprehensive online learning platform designed to facilitate course creation, management, and consumption. It allows instructors to upload educational content, manage students, and track earnings, while providing students with a seamless learning experience through video-based lessons and progress tracking.

## Core Goals
1. **Empower Instructors**: Provide tools for creating and organizing courses, including video uploads and content management.
2. **Engage Students**: Offer a structured learning environment with lessons, categories, and progress monitoring.
3. **Scalable Infrastructure**: Transition from a monolithic architecture to a microservices-based system to handle growth.
4. **Reliable Video Processing**: Utilize FFmpeg for robust video metadata extraction and processing.

## Key Features
- **User Management**: Role-based access control (Admin, Instructor, Student).
- **Course Catalog**: Categorized courses with objectives and requirements.
- **Learning Experience**: Video lessons with progress tracking.
- **Notifications**: In-app notifications for system and course updates.
- **Instructor Dashboard**: Statistics and payment management for content creators.
- **Cloud Storage**: Integration with AWS S3 for hosting media assets.
